# Lista de Tarefas

Aplicação de lista de tarefas desenvolvida com **HTML, CSS e JavaScript**, permitindo criar, visualizar e remover tarefas, com armazenamento dos dados no navegador utilizando `localStorage`.

O projeto foi desenvolvido com foco em praticar manipulação do DOM, eventos, criação dinâmica de elementos e persistência de dados no navegador.

##  Funcionalidades

*  Adicionar novas tarefas
*  Adicionar tarefas pressionando a tecla Enter
*  Criar elementos dinamicamente com JavaScript
*  Remover tarefas através do checkbox
*  Salvar tarefas no `localStorage`
*  Recuperar tarefas automaticamente ao recarregar a página
*  Interface responsiva para dispositivos móveis

##  Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* LocalStorage API
